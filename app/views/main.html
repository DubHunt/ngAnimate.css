ngAnimate.css

An extremely customisable and easy to use css library for animating AngularJS

How to use ...

Firstly, include the AngularJS ngAnimate module in your project:
  • Add the ngAnimate.js script
  • Reference a dependency to the 'ngAnimate' module in your app
Then finally include the simple css file or the customisable css file

Play around with the filter below to see the what animations are possible
To quickly see some smooth animations select a recommended (R) animation.
If you want to design your own then build up an animation by selecting a combination of:
  • All (A) - sets the animation for add, remove and move
  • Add (A) - sets the animation for when a row is added
  • Remove (R) - sets the animation for when a row is removed
  • Move (M) - sets the animation for when a row is moved
When you are happy with the animation just add the classes in the 'Classes applied' section to the element you are animating
